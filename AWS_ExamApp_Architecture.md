    {

      "Effect": "Allow",
      "Principal": {
        "Service": "cloudfront.amazonaws.com"
      },
      "Action": "s3:GetObject",
      "Resource": "arn:aws:s3:::aip-c01-practice-exam-jsmun/*",
      "Condition": {
        "StringEquals": {
          "AWS:SourceArn": "arn:aws:cloudfront::084828589246:distribution/E2LOJKWSZSCH8Y"
        }
      }
    }
  ]
}
```

#### 복습 규칙

| 규칙 | 설명 |
|------|------|
| **복습 대상** | 전날 틀린 문제 (신규 + 복습에서 틀린 것 모두 포함) |
| **복습 타이밍** | 다음 날 접속 시, 신규 문제 풀기 전에 권장 |
| **복습 성공** | 정답 맞추면 복습 목록에서 제거 |
| **복습 실패** | 또 틀리면 다음 날 복습에 다시 포함 |
| **복습 스킵** | 복습 건너뛰고 바로 신규 문제 풀기 가능 (강제 아님) |
| **복습 문제 수** | 일일 할당량에 포함하지 않음 (별도 카운트) |
| **최대 누적** | 복습 문제가 10개 초과 시 가장 오래된 것부터 우선 |

    └─ 오답 → 📖 내일 복습 대상
                  │
                  ├─ 복습에서 정답 → ✅ 완료
                  │
                  └─ 복습에서 오답 → 📖 다다음 날 복습 대상
                                        │
                                        └─ (반복, 맞출 때까지)
```

### 2.5 핵심 계산 로직

```javascript
// 일일 할당량 계산
function calculateDailyQuota(examDate, totalQuestions, completedCount) {
  const today = new Date();
  const exam = new Date(examDate);
  const daysLeft = Math.ceil((exam - today) / (1000 * 60 * 60 * 24));

  if (daysLeft <= 0) return totalQuestions - completedCount; // 시험 당일: 남은 거 전부
  
  const remaining = totalQuestions - completedCount;
  return Math.ceil(remaining / daysLeft);
}

// 예시
// 총 272문제, 42일 남음, 0문제 완료 → 하루 7문제
// 총 272문제, 42일 남음, 70문제 완료 → 하루 5문제 (남은 202 ÷ 42)
// 총 272문제, 10일 남음, 200문제 완료 → 하루 8문제 (남은 72 ÷ 10)
```

### 2.8.1 덤프 업로드 날짜 표시 기능

#### 왜 필요한가

AWS 자격증 시험은 문제가 주기적으로 업데이트됩니다. 오래된 덤프는 실제 시험과 다를 수 있으므로, 사용자가 덤프의 신선도를 판단할 수 있어야 합니다.

#### questions.js 메타데이터 추가

현재 `questions.js`는 바로 문제 배열로 시작합니다:

```javascript
// 현재 (변경 전)
const ALL_QUESTIONS = [ ... ];
```

메타데이터를 추가합니다:

```javascript
// 변경 후
const EXAM_META = {
  examCode: "ANS-C01",
  examName: "AWS Certified Advanced Networking Specialty",
  dumpDate: "2026-04-26",           // 덤프 업로드 날짜
  dumpVersion: "v2",                // 덤프 버전
  totalQuestions: 272,              // 총 문제 수
  source: "ExamTopics",            // 출처 (선택)
  lastUpdated: "2026-04-26",       // 마지막 수정일
  changelog: [                     // 변경 이력
    { date: "2026-04-26", note: "272문제 업로드" },
    { date: "2026-04-09", note: "초기 업로드 200문제" }
  ]
};

const ALL_QUESTIONS = [ ... ];
```

```javascript
function getDumpFreshness(dumpDate) {
  const now = new Date();
  const dump = new Date(dumpDate);
  const daysAgo = Math.floor((now - dump) / (1000 * 60 * 60 * 24));

  if (daysAgo <= 30) {
    return { label: `🟢 최신 (${daysAgo}일 전)`, color: '#4caf50' };
  } else if (daysAgo <= 90) {
    return { label: `🟡 보통 (${daysAgo}일 전)`, color: '#ff9800' };
  } else {
    return { label: `🔴 오래됨 (${daysAgo}일 전)`, color: '#f44336' };
  }
}
```
